## 🎯 About The Project

This is a **modern, responsive landing page** inspired by the spirit of **Samarkand** — blending ancient Silk Road heritage with cutting-edge IT creativity. Perfect for:

- Full-Stack Developer portfolios
- Tech agencies from Central Asia
- Personal branding (like Iqbolshoh Ilhomjonov)

## 🚀 Features

| Feature | Description |
|---------|-------------|
| 🎨 **Glassmorphism Design** | Modern backdrop-filter effects |
| 📱 **Fully Responsive** | Works perfectly on mobile, tablet & desktop |
| 🔢 **Animated Counters** | Numbers increase smoothly on scroll |
| 🌈 **Gradient Text & Icons** | Eye-catching color schemes |
| 🧭 **Smooth Navigation** | Scroll to sections without jumps |
| ⚡ **Interactive Elements** | Hover effects, dynamic badge change |
| 🎯 **Call-to-Action Buttons** | Ready for contact or download |

## 🛠️ Built With

- **HTML5** — Semantic structure
- **CSS3** — Flexbox, Grid, Custom properties, Animations
- **Vanilla JavaScript** — Intersection Observer, Dynamic counters, Smooth scroll
